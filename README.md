# Luxury Shopping Behavior Dashboard

## Project Overview
This project analyzes luxury retail shopping behavior to understand how customer demographics, product categories, ratings, and shipping options influence purchasing decisions. The analysis was conducted using Microsoft Excel and presented through an interactive dashboard designed for business insights.

## Objective
The goal of this project is to identify key drivers of purchase value and customer engagement in a luxury shopping context, helping retailers make informed decisions on inventory, marketing, and customer experience strategies.

## Dataset Description
The dataset contains customer shopping records with information on:
- Product categories and individual items
- Purchase amounts
- Customer gender and age group
- Review ratings
- Shipping types
- Purchase frequency and seasonal trends

## Data Preparation
The data was cleaned and prepared by:
- Removing duplicates and inconsistencies
- Standardizing category and item names
- Grouping customers by age and gender
- Creating calculated fields for performance comparison

## Analysis Approach
The analysis was carried out using:
- Pivot Tables for aggregation and comparison
- Trend analysis to evaluate item performance
- Category-level performance assessment
- Customer segmentation by demographics
- Shipping type evaluation based on purchase value

## Key Insights
- Clothing is the highest-performing product category by total purchase amount
- Blouse is the top-performing item across all products
- Male customers recorded higher overall review ratings
- Younger customers contribute the majority of positive reviews
- Free shipping generates the highest purchase value among all shipping options

## Dashboard Features
- Interactive slicers for size, season, gender, and purchase frequency
- Category and item performance comparison
- Customer demographic analysis
- Shipping type performance visualization

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel formulas
- Dashboard design and data visualization techniques

## Outcome
The dashboard provides a clear and interactive view of luxury shopping behavior, enabling stakeholders to quickly identify profitable products, valuable customer segments, and effective sales incentives.
